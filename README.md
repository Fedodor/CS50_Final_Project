# CS50_Final_Project
Video Demo: https://youtu.be/14pbbAfhqMw

## Stack
- FastAPI + pydantic, pydantic-settings, aiohttp
- React + axios, ant design, tailwind


### Instructions
#### Backend
- `python3 -m venv venv`
- `. venv/bin/activate`
- `pip install -r requirements.txt`
- `uvicorn src.main:app --reload`

#### Frontend
- `npm create vite@latest`
- `npm install`
- `npm run dev`
